# Bugs(zh-CN)

**修復良性 Bug 前請審慎評估將對代理人造成之影響**

Bug 暫時分類為以下三種 Tag
* 惡性
  * 正面效果無法發動，導致收益不如預期。
  * 實際效果低於文本效果
* 中性
  * 效果錯誤，失去預期效果但帶來額外收益
* 良性
  * 負面效果無法發動，導致收益優於預期。
  * 實際效果優於文本效果

敘述為格式如下
```
## Doll/Item Name
* Skill Set
  * Tag
  * Expected
  * Actual
  * Comment(Optional)
```
